# Machine Learning Environment (Phase 2)

This directory is strictly reserved for the Python code responsible for training the Machine Learning model. 
As per the project rules, **no Java code** should be placed here.

## Structure
- `/datasets/`: Local storage for telemetry CSVs generated by UR-OS. (Note: Large CSVs are ignored by `.gitignore`).
- `/models/`: Local storage for exported trained models (e.g., decision tree rules, JSON, `.pkl`).
- Scripts for dataset generation, training (`train_model.py`), and evaluation will be located in this root directory.

## Workflow
1. Run UR-OS in data-collection mode to generate a CSV.
2. Place the CSV in `/datasets/`.
3. Run the Python training script.
4. Export the lightweight model to `/models/` or directly into the Java source code structure.
